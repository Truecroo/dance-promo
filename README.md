# Dance Promo

Промо-игра для ивента на базе механики Dance Dance Revolution.
Стрелки летят с 4 сторон к центру, где стоит танцор. Жми в нужный момент —
чем длиннее комбо, тем круче он танцует и тем больше очков.

## Запуск

```bash
python3 -m http.server 5599
# открыть http://localhost:5599
```

Чистый HTML/CSS/JS, без сборки и зависимостей.

## Управление

- Десктоп: стрелки на клавиатуре ← ↓ ↑ →
- Мобилки: тач-кнопки внизу экрана

## Что менять под ивент (`js/config.js`)

| Поле | Что задаёт |
|------|-----------|
| `arrow.speed`, `spawn.*` | сложность / темп |
| `trackDuration` | длина раунда (под музыку) |
| `scoreTiers` | очки по уровню комбо |
| `moods` + `dancerSprite` | пороги и спрайты «настроения» танцора |
| `dirColors` | цвета стрелок |
| `endings` | финальные тексты по очкам |

## Плейсхолдеры (заменить на ассеты)

- **Танцор** — эмодзи в `dancerSprite`. Заменить на спрайт-лист / Lottie / GIF
  (DOM-элемент `#dancer`, анимации в `css/style.css`).
- **Стрелки** — рисуются треугольниками на canvas (`drawArrowGlyph`).
  Заменить на картинки при необходимости.
- **Музыка/звук** — пока нет; добавить `<audio>` и завязать `trackDuration` на длину трека.
